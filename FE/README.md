# BringAuto Fleet Front-End

## Project setup
Environment variables to setup:
* APP_VUE_GRAPHQL_URL
* APP_VUE_SENTRY_URL

Node packages setup:
```
npm install
```

### Compiles and hot-reloads for development

npm run serve
